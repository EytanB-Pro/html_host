Vercel Serverless User Creator

Fill in a name and password, then submit to post to the serverless `/api/users` endpoint.

Submit the form to create a user through Flask.
What this is

A basic static HTML page with inline CSS and JavaScript. Perfect for Vercel's front-end preview.

How to use it

Run the Flask app locally, then open this page and submit the form to create a user.

Backend

This page sends JSON to `/api/users` and receives status messages from the Vercel serverless function.
